# rRedis

This is a learning project to implement a Redis client in Rust. And we will follow [Build Your Own Redis with C/C++](https://build-your-own.org/redis/#table-of-contents). Though the original project is in C/C++, we will implement it in Rust to learn Rust and also to see how the implementation differs in Rust.

## Learning Topics

- Understanding the basic structure of Redis and how it works.
- The RESP protocol and how to implement it in Rust.

## Tech Stack

- Rust: The programming language used for the implementation.
- Cargo: The Rust package manager and build system.
- Git: For version control and collaboration.
